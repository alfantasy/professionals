Настройка интерфейсов

Первый интерфейс

  • Нажмите "Изменить" на существующем интерфейсе
  • Назначьте описательное имя
  • Добавьте в соответствующую зону безопасности

Второй интерфейс

  • Нажмите "Добавить интерфейс"
  • Настройте параметры IP
  • Назначьте зону безопасности

Настройка OSPF

Важно: Отключите все флажки перед настройкой
  1. Включите протокол OSPF
  2. Добавьте интерфейсы с указанием:
    • Имени интерфейса
    • Имени зоны/области
    • Стоимости маршрута

Доступ в интернет

Создание пользователя

  • Перейдите: Пользователи → Учетные записи
  • Нажмите иконку добавления учетной записи
  • Укажите:
    • Имя: network
    • Логин: network

Авторизация

  • Перейдите: Пользователи → Авторизация
  • Укажите:
    • Пользователь: network
    • Подсеть: [IP/Маска]
